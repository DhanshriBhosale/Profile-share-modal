# Profile Share Modal 🚀

A simple and responsive **Profile Card with Share Modal** built using HTML, CSS, and JavaScript.

## 🔥 Features

* Profile card UI
* Share modal popup
* Social media icons (Facebook, Twitter, Instagram, WhatsApp, Telegram)
* Smooth animations & overlay effect

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
└── assets/
```

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

## 📸 Preview

Clean UI with animated background and share modal popup.
